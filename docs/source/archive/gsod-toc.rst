=============================
GSoD -- Google Season of Docs
=============================

**Google Season of Docs** is an international annual program giving
technical writers an opportunity to contribute to open source projects.It
fosters the open source contributions of technical writers. GSoD was
started by Google in 2019. More about of GSoD -
<https://developers.google.com/season-of-docs>_
